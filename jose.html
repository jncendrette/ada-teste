git log --oneline: Exibe cada commit em uma linha com um resumo conciso (hash e mensagem). git log --graph: Mostra um gráfico ASCII que representa a árvore de commits, ajudando a visualizar o histórico de ramificações e merges. git log --author="Nome": Filtra os commits feitos por um autor específico. git log --since="2023-01-01": Exibe commits feitos após uma data específica.
